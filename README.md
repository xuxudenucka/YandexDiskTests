# Автотесты для API Яндекс Диска

Проект содержит пример автоматизации базовых операций с ресурсами (папками) через Yandex Disk REST API.

## Стек технологий

- **Java 11+**: основной язык программирования.
- **JUnit 5**: фреймворк для написания и запуска тестов.
- **Java HttpClient**: встроенный клиент для отправки HTTP-запросов.

## Структура проекта

- `src/main/java`: логика API-клиента и конфигурации.
- `src/test/java`: функциональные тесты, покрывающие методы GET, POST, PUT и DELETE.

## Настройка и запуск

Для работы тестов необходим OAuth-токен. В целях безопасности токен не хранится в коде, а считывается из переменной окружения.

### Шаги настройки

1. Получите токен на [Яндекс.Полигоне](https://yandex.ru/dev/disk/poligon/).

2. Установите переменную окружения `YANDEX_DISK_TOKEN`:

    - **В терминале (Linux/macOS)**:
      ```bash
      export YANDEX_DISK_TOKEN='ваш_токен'
      ```

    - **В Windows (PowerShell)**:
      ```powershell
      $env:YANDEX_DISK_TOKEN='ваш_токен'
      ```

    - **В настройках IDE (IntelliJ IDEA)**: добавьте переменную в Run/Debug Configurations.

3. Запустите тесты через Gradle или вашу IDE:

   ```bash
   ./gradlew test
   ```
